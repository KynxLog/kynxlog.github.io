CSS3变换教程

掌握CSS3变换，为网页元素添加动态效果和三维体验

1. 什么是CSS3变换？

CSS3变换（Transform）允许我们对元素进行移动、旋转、缩放和倾斜等操作，而不会影响文档的正常流。它可以创建2D和3D效果，为网页增添动态和交互性。

/* 基本语法 */
element {
  transform: none | <transform-function>;
}

其中 <transform-function> 可以是以下一个或多个函数：

  • 2D变换函数：translate(), rotate(), scale(), skew()
  • 3D变换函数：translate3d(), rotate3d(), scale3d()

注意：多个变换函数可以同时应用，它们会按照从右到左的顺序执行。例如：transform: rotate(30deg) translate(100px); 会先平移元素，然后再旋转。

2.1 平移（Translate）

平移变换将元素从其当前位置移动到新位置，不影响其他元素布局。

  • translateX(tx)
    • 仅水平移动。
  • translateY(ty)
    • 仅垂直移动。
  • translate(tx, ty)
    • tx: 水平方向的移动距离。正值向右，负值向左。
    • ty: 垂直方向的移动距离。正值向下，负值向上。如果未提供，则默认为 0。
    • 单位：常用 px, em, % 等。
CSS语法
/* 示例 */
.element {
  transform: translate(50px, 20px); /* 沿X轴移动50px，Y轴移动20px */
  transform: translate(50%); /* 沿X轴移动50% */
}

注意：

    % 是相对于元素自身的宽度(translateX)或高度(translateY)的，而不是父容器。

2.2 倾斜（Skew）

倾斜变换将元素沿着X轴和/或Y轴倾斜一定角度。

  • skewX(ax)
    • 仅沿X轴倾斜。
  • skewY(ay)
    • 仅沿Y轴倾斜。
  • skew(ax, ay)
    • ax: 水平方向（X轴）的倾斜角度。
    • ay: 垂直方向（Y轴）的倾斜角度。如果未提供，则默认为 0。
    • 单位：deg, rad 等。
CSS语法
/* 示例 */
.element {
  transform: skew(20deg); /* 沿X轴倾斜20度 */
  transform: skew(20deg, 10deg); /* 同时沿X轴倾斜20度，Y轴倾斜10度 */ }

2.3 缩放（Scale）

缩放变换改变元素的尺寸，可以等比例或非等比例缩放。scale()函数的参数是一个无单位的数值，表示缩放倍数。

  • scaleX(sx)
    • 仅水平缩放。
  • scaleY(sy)
    • 仅垂直缩放。
  • scale(sx, sy)
    • sx: 水平方向的缩放倍数。
    • sy: 垂直方向的缩放倍数。如果未提供，则默认与 sx 相同，即等比例缩放。
    • 值：无单位数字。1 表示原始大小，2 表示放大一倍，0.5 表示缩小一半，-1 表示翻转并缩放。
CSS语法
/* 示例 */
.element {
  transform: scale(1.5); /* 放大到1.5倍 */
  transform: scale(1.5, 0.8); /* X轴放大1.5倍，Y轴缩小到0.8倍 */
}

2.4 旋转（Rotate）

旋转变换将元素围绕变换原点旋转指定角度。

  • rotate(angle)
    • angle: 旋转的角度。
    • 单位：常用 deg (度)，也可以是 rad (弧度)、grad (百分度)、turn (圈)。
    • 正值：顺时针旋转。负值：逆时针旋转。
CSS语法
/* 示例 */
.element {
transform: rotate(30deg); /* 顺时针旋转30度 */
transform: rotate(-45deg); /* 逆时针旋转45度 */
}

2.5 变换原点 (transform-origin)

transform-origin 属性用于更改元素变换的原点，即元素围绕哪个点进行变换。默认情况下，变换的原点位于元素的中心(50% 50%)。

CSS语法
/* 语法格式： */
transform-origin: x-axis y-axis z-axis;

/* 示例 */
.element {
  transform-origin: center; /* 中心点 */
  transform-origin: left top; /* 左上角 */
  transform-origin: 50px 50px; /* 具体坐标 */
  transform-origin: 0% 0%; /* 百分比 */
  transform-origin: left top 50px; /* 3D变换中的Z轴原点 */
}

参数说明：

  • x-axis：定义X轴上的原点位置，可选值：left, center, right, 长度值(px, em等), 百分比(%)
  • y-axis：定义Y轴上的原点位置，可选值：top, center, bottom, 长度值, 百分比
  • z-axis：定义Z轴上的原点位置（仅用于3D变换），长度值

关键字表示

关键字 描述
left 原点为元素左边中间位置，同 left center
right 原点为元素右边中间位置，同 right center
top 原点为元素上边中间位置，同 top center
bottom 原点为元素下边中间位置，同 bottom center
center 原点为元素中间位置，同 center center
top left 原点为元素左上角，相当于坐标 (0 0)
top right 原点为元素右上角
left bottom 原点为元素左下角
right bottom 原点为元素右下角

2.6 组合变换

可以同时应用多个变换函数，注意变换的顺序会影响最终结果。

CSS语法
/* 先平移再旋转 */
transform: translate(100px) rotate(45deg);
/* 先旋转再平移（结果不同） */
transform: rotate(45deg) translate(100px);

3.1 3D空间与景深（Perspective）

要创建3D效果，首先需要建立一个具备透视（Perspective）的3D场景（舞台）。

  • perspective 属性定义了三维空间（3D space）的深度，更具体地说，它设定了观察者（用户的眼睛）与z=0平面（屏幕）之间的距离，它的核心作用是创建“近大远小”的透视效果。它的值是一个长度单位（如 px, em），表示这个距离。
  • transform-style 属性决定了一个元素的孩子（子元素）是位于这个元素本身所定义的3D空间中，还是被扁平化（压扁）在这个元素的2D平面内。

可以把整个结构想象成一个摄影棚：

  • 父元素：是整个摄影棚。
    • perspective: 800px; 是决定摄影师（用户眼睛）使用什么镜头（透视强度）。
    • transform-style: preserve-3d; 是决定这个摄影棚是一个真实的3D舞台，而不是一个2D的幕布。
  • 子元素：是舞台上的演员。
    • transform: rotateY(45deg) translateZ(100px); 是演员在舞台上的位置和姿势。
CSS语法
.container {
  perspective: 800px; /* 在父元素上设置景深 */
  transform-style: preserve-3d; /* 将子元素置于父元素创建的 3D 空间内 */
}

3.2 3D位移

3D位移允许元素在Z轴上移动，创建远近效果。

CSS语法
/* 语法 */
transform: translateX(x) - 仅沿X轴移动元素（横向位移，左右方向）
transform: translateY(y) - 仅沿Y轴移动元素（纵向位移，上下方向）
transform: translateZ(z) - 仅沿Z轴移动元素（纵深位移，前后方向）
transform: translate3d(tx, ty, tz) - 同时指定X、Y、Z三个方向的位移

/* 示例 */
.element {
  transform: translateZ(200px);
}

.element {
  transform: translate3d(50px, -10px, 200px);
}

3.3 3D旋转

3D旋转允许元素围绕X、Y或Z轴旋转。

CSS语法
/* 围绕X轴旋转 */
transform: rotateX(45deg);

/* 围绕Y轴旋转 */
transform: rotateY(45deg);

/* 围绕Z轴旋转（与2D旋转相同） */
transform: rotateZ(45deg);

/* 围绕自定义轴旋转 */
transform: rotate3d(x, y, z, angle);

/* 示例 */
.element {
  transform: rotateZ(45deg);  /* 围绕Z轴旋转45度 */
}

.element {
  transform: rotate3d(1, 1, 0, 45deg);  /* x=1, y=1, z=0 定义了一个在xy平面上的对角线轴 */
}

rotate3d(x, y, z, angle)参数说明：

  • 作用：使元素围绕 自定义的三维轴 旋转，更灵活地控制旋转方向。
  • x/y/z：数值（0~1 之间），表示旋转轴在对应坐标轴上的分量（0 表示不沿该轴，1 表示完全沿该轴）。
  • angle：旋转角度（如 45deg）。

3.4 3D变换原点

3D变换中，transform-origin属性同样适用，但可以接受三个值，包括Z轴偏移。

CSS语法
/* 默认值 */
transform-origin: 50% 50% 0;
/* 自定义3D变换原点 */
transform-origin: 0% 0% 20px;
transform-origin: 30% 60% 50px;

3D变换演示案例：卡片翻转效果

使用3D变换创建卡片翻转效果，需要设置透视(perspective)和3D变换样式。

正面

3D卡片翻转效果

CSS语法
<style>
/* 步骤1: 设置景深与 3D 空间 */
.container {
  perspective: 1000px;
  transform-style: preserve-3d;
}
/* 步骤2: 卡片的初始状态 */
.card {
  width: 200px;
  height: 300px;
  padding: 10px;
  background-color: skyblue;
  border-radius: 8px;
  box-shadow: 0 4px 16px rgba(0, 0, 0, 0.1);
  cursor: pointer;
}
/* 步骤3: 卡片的结束状态-翻转180度 */
.card:hover {
  transform: rotateY(180deg);
}
/* 步骤4: 配置过渡属性 */
.card {
  transition: transform 1s;
}
</style>
<body>
  <div class="container">
    <div class="card">正面</div>
  </div>
</body>

6. 综合案例：图片放大效果

下面是一个使用CSS3变换实现的图片放大效果，当鼠标悬停在图片上时，图片会平滑放大，同时卡片整体会有轻微的浮动效果。

参考代码：

HTML代码
<div class="poem-card">
  <div class="poem-image">
    <img src="nature.jpg" alt="春夜喜雨">
  </div>
  <div class="poem-content">
    <h4>春夜喜雨</h4>
    <p>好雨知时节，当春乃发生。</p>
    <p>随风潜入夜，润物细无声。</p>
  </div>
</div>
CSS代码
/* 步骤1: 卡片的初始状态 */
.poem-card {
  width: 300px;
  background: white;
  border-radius: 12px;
  box-shadow: 0 10px 25px rgba(0, 0, 0, 0.12);
  overflow: hidden;
  cursor: pointer;
}

.poem-image {
  height: 300px;
  overflow: hidden;
}

.poem-image img {
  width: 100%;
  height: 100%;
  object-fit: cover;
}

.poem-content {
  padding: 20px;
  text-align: center;
  background: #f8f5eb;
}

.poem-content h3 {
  color: #8c7851;
  margin: 0;
  font-size: 1.4rem;
}

.poem-content p {
  color: #7a6a52;
  font-size: 1rem;
  line-height: 1.6;
}

/* 步骤2：卡片的结束状态-图片放大效果 */
.poem-card:hover {
  box-shadow: 0 15px 35px rgba(0, 0, 0, 0.15);
  transform: translateY(-5px);
}

.poem-image img:hover {
  transform: scale(1.25);
}

/* 步骤3: 配置过渡属性 */
.poem-card {
  transition: box-shadow 0.4s ease;
}

.poem-image img {
  transition: transform 0.6s ease;
}

技术要点：

  • 使用transform: scale(1.25)实现图片放大效果
  • 使用transition属性为变换添加平滑过渡
  • 使用overflow: hidden隐藏图片放大时超出容器的部分
  • 使用object-fit: cover确保图片比例正确
