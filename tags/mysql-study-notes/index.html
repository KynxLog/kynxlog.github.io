数据库操作与Navicat 17使用

MySQL组件安装过程

MySQL是一个管理系统，所以在安装的时候需要一个可以方便与其交互的软件，所以必须要有Navicat这样一个图形化界面的软件，更便于对数据库进行操作 安装C++运行库，MySQL需要使用（安装完成后关闭即可） 安装MySQL 双击运行安装程序，直至出现这个界面： 点击Next（如果对于英文界面熟悉，且需要更换数据存储目录可以选择Custom自定义安装）： 点击Execute，等待出现如下界面： 点击Next： 继续点击Next： 继续点击Next： 继续点击Next： 出现这个界面之后就需要输入管理员账户密码，也就是Root密码，输入完成之后点击Next： 点击Next： 点击Next（这里调整了数据存储位置，与图片不一样不用在意）： 点击Execute，出现这个界面就是已经安装完成，点击Finnish即可： 添加环境变量： 找到安装文件目录，一般在C:\Program Files\MySQL\MySQL Server 8.0\bin，自定义的除外，复制其安装路径 打开环境变量编辑页面 右键此电脑，选择属性 找到高级系统设置 选择环境变量 上方的用户变量和下方的系统变量都可以，在其中找到Path双击即可 点击新建并将刚刚复制的路径粘贴进去 点击确定即可 安装Navicat 17 双击执行安装程序，无脑下一步即可 右键桌面上程序图标，选择打开文件所在位置，Windows11可以在显示更多选项中找到 将winmm.dll应用程序扩展放入 即可开始使用
