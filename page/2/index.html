数据库操作与Navicat 17使用

使用命令操作数据库 管理系统更像是没有桌面的Linux操作系统，需要使用命令对其进行控制与操作。 使用cmd连接MySQL数据库并进行操作 使用Windows徽标键+R 输入cmd并回车，会打开一个新的窗口 以 mysql -hIP地址(如127.0.0.1) -P端口号(如3306) -u用户名(如root) -p密码(如root) 这样的格式键入，即可连接本地数据库 进入之后就可以对数据库进行增删改查的操作 使用Navicat 17连接管理MySQL 在刚刚使用命令行对数据库进行操作的时候，不管是使用create还是drop对数据库进行更改的时候查看起来其实非常不方便，每次都要show databases，所以可以使用Navicat的图形界面更便于管理。 打开Navicat 里面什么都没有，所以我们需要新建一个和本地数据库的连接。点击左上角的连接，会有一个弹出窗口 选中MySQL之后点击下一步 在这里按照提示填入信息（如上图），点击测试连接，查看是否可以正常连接 提示可以正常连接之后点击确定即可。 在连接的右边有一个新建查询，点击后即可出现一下界面 在出现的地方就可以使用命令操作数据库，以create database new01为例 单击运行 提示成功之后，左侧数据库列表并没有显示，所以需要右键刷新 刷新后即可看到 与create步骤一样，以下是drop执行完之后
