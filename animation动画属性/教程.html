CSS3动画教程

掌握CSS3动画，为网页元素添加流畅的动态效果和交互体验

1. CSS3动画简介

CSS3动画（Animation）通过关键帧控制元素的样式变化过程，创建复杂的动画效果。与过渡（transition）不同，动画不需要触发事件，可以自动播放，并且可以定义多个中间状态。

2. 实现CSS3动画的步骤

步骤 1: 定义关键帧

使用 @keyframes 规则创建动画序列，通过定义多个关键帧实现元素在不同状态之间的平滑过渡。

步骤 2: 应用动画到元素

使用 animation 属性将动画绑定到元素，至少需要指定动画名称和持续时间。

步骤 3: 配置动画属性

配置动画的延迟、播放次数、方向、时间函数等参数。

3. CSS3 动画组成部分

CSS3 动画由以下两个部分组成：

  • @keyframes规则：定义动画的规则和内容。它就像是动画的“剧本”，规定了元素在动画过程中不同时间点应该呈现的样式状态（即关键帧）。
  • animation 动画属性：定义动画的行为和控制。它就像是“导演”，负责挑选演员（元素）、决定用哪个剧本（动画名称）、演出速度、次数等。

3.1 关键帧 @keyframes

@keyframes规则是CSS动画的核心，它定义了动画从开始到结束的整个序列。可以在动画序列中设置多个关键帧，每个关键帧描述元素在特定时间点应用的样式。

CSS语法
@keyframes animation-name {
  0% {
    /* 起始状态样式 */
  }
  50% {
    /* 中间状态样式 */
  }
  100% {
    /* 结束状态样式 */
  }
}

/* 或者使用from和to关键字 */
@keyframes animation-name {
  from {
    /* 起始状态样式 */
  }
  to {
    /* 结束状态样式 */
  }
}

注意：

  • 关键帧选择器可以使用百分比（0%到100%）或from（0%）和to（100%）关键字
  • 如果0%或100%未指定关键帧，浏览器将使用元素的默认样式
  • 关键帧中的样式将覆盖元素原有的样式

定义进度条动画关键帧：

CSS代码 
/* 步骤1：定义关键帧 */
@keyframes progress {
    from {
        width: 0%;
    }
    to {
        width: 100%;
    }
}

3.2 animation动画属性

CSS动画通过一系列属性来控制动画的播放方式，这些属性可以单独设置，也可以使用animation简写属性。

属性 描述 示例值
animation-name 指定要应用的@keyframes动画名称 slide, fade, progress
animation-duration 动画完成一个周期所需的时间 2s, 500ms
animation-timing-function 动画的速度曲线 ease, linear, ease-in-out ...
animation-delay 动画开始前的延迟时间 1s, 200ms
animation-iteration-count 动画播放次数 1, 3, infinite
animation-direction 动画播放方向 normal, reverse, alternate, alternate-reverse
animation-fill-mode 动画执行前后如何应用样式 none, forwards, backwards, both
animation-play-state 动画播放状态 running, paused

定义进度条动画的animation属性：

CSS代码 
/* 步骤2：将动画应用到元素 */
.progress-bar {
    animation-name: progress;
    animation-duration: 5s;
}

/* 步骤3：配置动画属性 */
.progress-bar {
    animation-timing-function: linear;
    animation-delay: 1s;
    animation-iteration-count: infinite;
}

3.3 animation简写属性

CSS animation 属性是多个动画属性的简写形式，可以一次性设置所有动画参数，使代码更简洁、易读。

基本语法
/* 语法格式 */
animation: name duration timing-function delay iteration-count direction fill-mode play-state;

/* 实际应用示例 */
animation: progress 5s linear 1s infinite;

属性顺序记忆技巧

记住这个顺序的一个简单方法是使用助记符：

N（名称） D（持续时间） T（时间函数） D（延迟） I（迭代次数） D（方向） F（填充模式） P（播放状态）

口诀："名字持续怎么动，延迟几次往哪动，填充状态别漏掉"

注意：

  • 简写属性中，animation-duration和animation-delay的时间值顺序很重要：第一个时间值总是duration，第二个是delay
  • 不是所有属性都必须指定，但animation-name和animation-duration是必需的
  • 可以同时为元素应用多个动画，用逗号分隔：animation: slide 2s ease, fade 3s linear;

4. 练一练：淡入淡出动画

淡入淡出动画是网页设计中常用的入场和离场效果，可以平滑地显示或隐藏元素。

淡入淡出动画

CSS代码 
/* 淡入淡出动画 */
@keyframes fade-in-out {
    0% {
        opacity: 0;
        transform: translateY(-20px);
    }
    50% {
        opacity: 1;
        transform: translateY(0);
    }
    100% {
        opacity: 0;
        transform: translateY(20px);
    }
}

.fade-in-out {
    animation: fade-in-out 5s ease-in-out 1s infinite;
}

参考代码：

CSS代码
/* 滚动动画 */
@keyframes scrollPoem {
  0% {
    transform: translateY(0);
  }
  100% {
    transform: translateY(-25%);
  }
}
/* 应用动画 */
.poem-content {
  animation: scrollPoem 15s linear 1s forwards infinite;
  animation-play-state: running; /* 默认运行动画 */
}
.poem-card:hover .poem-content {
  animation-play-state: paused; /* 暂停动画 */
}

技术要点：

  • 使用@keyframes创建文字滚动动画效果
  • 使用animation: scrollPoem 15s linear infinite实现无限循环滚动
  • 使用overflow: hidden隐藏超出容器的内容
  • 使用transform: translateY()实现垂直方向的移动
  • 使用animation-play-state: paused实现鼠标悬停时诗词暂停滚动
