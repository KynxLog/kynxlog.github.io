CSS3过渡动画教程

掌握CSS3过渡动画，为网页添加流畅的交互效果

1. 什么是过渡动画？

CSS3过渡（Transition）允许元素在不同状态之间平滑地变化，而不是瞬间改变。它通过补间动画实现，在初始状态和结束状态之间自动生成中间状态。

2. 操作步骤

1定义初始状态

设置元素的初始样式属性，包括位置、尺寸、颜色等。

2定义过渡后结束状态

使用伪类（:hover, :focus, :active）来定义变化后的状态。

3配置过渡属性

使用transition属性指定哪些属性需要过渡、持续时间、缓动函数和延迟时间。

3. 过渡动画的语法

CSS语法
/* 简写形式 */
transition: property duration timing-function delay;

/* 分开书写形式 */
transition-property: all | property1, property2, ...;
transition-duration: time;
transition-timing-function: ease | linear | ease-in | ease-out | ease-in-out | cubic-bezier(n,n,n,n);
transition-delay: time;
属性 描述 默认值 是否必须
transition-property 指定应用过渡的CSS属性名称 all（所有可过渡发展） 否
transition-duration 指定过渡完成所需的时间 0s 是
transition-timing-function 指定过渡的速度曲线 ease（先慢后快再慢） 否
transition-delay 指定过渡开始前的延迟时间 0s 否

3.1 transition-property：定义 “过渡的属性”

该属性指定哪些 CSS 属性会触发过渡效果（如宽度、颜色、透明度等）。只有被指定的属性，在值发生变化时才会平滑过渡；未指定的属性会瞬间变化。

核心细节：

  • 默认值：all（表示所有可过渡的 CSS 属性，值变化时都会触发过渡）。
  • 常用取值：
    • 具体属性名：如 width、height、background-color、opacity、transform 等（需确保属性本身支持过渡，如 display: none 不支持过渡）。
    • none：禁止任何属性的过渡效果。
  • 语法：transition-property: 属性1, 属性2, ...;（多个属性用逗号分隔）。

示例：

HTML/CSS代码
<style type="text/css">
/* 第1步：定义元素的初始状态 */
.box {
  width: 100px;
  height: 100px;
  background: tomato;
}
/* 第2步：定义元素过渡后的结束状态 */
.box:hover {
  border-radius: 50%;
}
/* 第3步：配置过渡属性 */
.box {
  transition-property: border-radius;
}
</style>
<body>
  <div class="box"></div>
</body>

3.2 transition-duration：定义 “过渡的时长”

该属性指定过渡效果从 “初始状态” 到 “结束状态” 的总耗时，是过渡效果的 “基础时间单位”，必须设置（默认值为 0s，不显示过渡）。

核心细节：

  • 默认值：0s（无过渡，属性瞬间变化）。
  • 取值格式：以时间单位 s（秒）或 ms（毫秒）表示，如 0.5s、300ms（注意：值为 0 时可省略单位，但非 0 必须带单位）。
  • 语法：transition-duration: 时长1, 时长2, ...;（多个时长与 transition-property 一一对应，数量不一致时会循环匹配）。

示例：

HTML/CSS代码
<style type="text/css">
/* 第1步：定义元素的初始状态 */
.box {
  width: 100px;
  height: 100px;
  background: tomato;
}
/* 第2步：定义元素过渡后的结束状态 */
.box:hover {
  border-radius: 50%;
}
/* 第3步：配置过渡属性 */
.box {
  transition-property: border-radius;
  transition-duration: 1s;
}
</style>
<body>
  <div class="box"></div>
</body>

3.3 transition-timing-function：定义 “过渡的速度曲线”

该属性控制过渡过程中 “速度的变化规律”，决定了过渡是 “匀速”“先快后慢” 还是 “先慢后快” 等效果，核心是通过贝塞尔曲线（Cubic Bezier） 实现。

常用取值（内置曲线）
值 描述
ease 默认值，慢速开始，然后变快，然后慢速结束
linear 匀速运动
ease-in 慢速开始，然后加速
ease-out 快速开始，然后减速
ease-in-out 慢速开始和结束，中间加速
cubic-bezier(x1,y1,x2,y2) 自定义速度曲线，使用贝塞尔曲线

拓展：贝塞尔曲线理解与可视化

贝塞尔曲线是计算机图形学中使用的一种参数曲线，由法国工程师皮埃尔·贝塞尔于1962年广泛发表，用于汽车车身设计。

在CSS中，我们使用三次贝塞尔曲线（Cubic Bezier）来描述动画的速度曲线，它由四个点定义：

  • P0：起点 (0, 0)
  • P1：第一个控制点 (x1,y1)
  • P2：第二个控制点 (x2,y2)
  • P3：终点 (1, 1)

CSS中的cubic-bezier()函数只接受四个参数：cubic-bezier(x1,y1,x2,y2)，因为起点和终点固定为(0,0)和(1,1)。

如何理解贝塞尔曲线？

贝塞尔曲线描述了动画过程中时间与进度的关系：

  • X轴：表示时间（从0%到100%）
  • Y轴：表示动画进度的完成度（从0%到100%）

曲线的斜率代表了动画的速度：

  • 陡峭的曲线部分 = 快速变化
  • 平缓的曲线部分 = 缓慢变化
  • 水平线 = 暂停/无变化
  • 向下的曲线 = 反向运动（在某些情况下）

常见的时间函数曲线：

/* 默认值，慢快慢 */
transition-timing-function: ease;
/* 相当于 */
transition-timing-function: cubic-bezier(0.25, 0.1, 0.25, 1);

/* 线性 */
transition-timing-function: linear;
/* 相当于 */
transition-timing-function: cubic-bezier(0, 0, 1, 1);

/* 渐入 */
transition-timing-function: ease-in;
/* 相当于 */
transition-timing-function: cubic-bezier(0.42, 0, 1, 1);

/* 渐出 */
transition-timing-function: ease-out;
/* 相当于 */
transition-timing-function: cubic-bezier(0, 0, 0.58, 1);

/* 渐入渐出 */
transition-timing-function: ease-in-out;
/* 相当于 */
transition-timing-function: cubic-bezier(0.42, 0, 0.58, 1);

3.4 transition-delay：定义 “过渡的延迟时间”

该属性指定过渡开始前的等待时间，可实现 “延迟触发过渡” 的效果（如 hover 后等待 0.5 秒再开始动画）。

核心细节：

  • 默认值：0s（属性变化后立即开始过渡）。
  • 取值格式：同 transition-duration，用 s 或 ms 表示（可设为负数，此时过渡会从中间状态开始，如 -0.5s 表示跳过前 0.5 秒的过渡）。
  • 语法：transition-delay: 延迟1, 延迟2, ...;（与 transition-property 一一对应，规则同 duration）。

示例：

hover 后延迟 1 秒再开始过渡：

HTML/CSS代码
<style>
/* 第1步：定义元素的初始状态 */
.box {
  width: 100px;
  height: 100px;
  background: tomato;
}
/* 第2步：定义元素过渡后的结束状态 */
.box:hover {
  border-radius: 50%;
}
/* 第3步：配置过渡属性 */
.box {
  transition-property: border-radius;
  transition-duration: 1s;
  transition-timing-function: ease-in;
  transition-delay: 1s;
}
</style>
<body>
  <div class="box"></div>
</body>

3.6 定义多个过渡动画

每个过渡动画间隔1秒依次执行。

HTML/CSS代码
<style>
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  opacity: 1;
/*
动画1：宽度变化
动画2：延迟1s后，高度发生变化
动画3：延迟2s后，透明度发生变化
*/
  transition: width 1s linear 0s, height 1s ease 1s, opacity 1s linear 2s;
}
/* 鼠标滑动后，改变宽，高，透明度 */
.box:hover {
  width: 200px;
  height: 200px;
  opacity: 0.2;
}
</style>
<body>

</body>

4. 综合案例：古诗词卡片

下面是一个使用CSS3过渡效果的古诗词卡片综合案例。将鼠标悬停在卡片上，可以看到多种过渡效果的应用。

静夜思

李白 · 唐

床前明月光，疑是地上霜。

举头望明月，低头思故乡。

古诗词卡片过渡效果代码：

HTML/CSS代码
<style>
/* 全局样式 */
body {
  font-family: "宋体", serif;
  font-size: 14px;
  color: #333;
  background-color: #f5f0e6;
  margin: 0;
}
/* 第1步：卡片的初始状态样式 */
.poem-card {
  width: 280px;
  margin: 50px auto;
  padding: 20px;
  border: 1px solid #e0d6c2;
  border-radius: 8px;
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
  background-color: #fff;
}
.poem-card h3 {
  margin-top: 0;
  border-bottom: 1px solid #E0D6C2;
  padding-bottom: 10px;
  color: #8b5a2b;
}
.poem-card .author {
  margin: 5px 0 15px;
  color: #d4a76a;
  font-size: 12px;
}
.poem-card .poem-content {
  line-height: 1.8;
}
/* 第2步: 卡片的结束状态样式 */
.poem-card:hover {
  background-color: #f9f5e9;
  box-shadow: 0 6px 12px rgba(0, 0, 0, 0.15);
}
.poem-card:hover .author {
  color: #8b5a2b;
}
/* 第3步: 配置过渡属性 */
.poem-card {
  transition: all 1s ease;
}
.poem-card .author {
  transition: color 1s ease;
}
</style>
<body>
  <div class="poem-card">
    <h3>静夜思</h3>
    <p class="author">李白·唐</p>
    <div class="poem-content">
      <p>床前明月光，疑是地上霜。</p>
      <p>举头望明月，低头思故乡。</p>
    </div>
  </div>
</body>

关键代码说明：

  • transition-property：指定哪些CSS属性需要过渡效果
  • transition-duration：设置过渡动画的持续时间
  • transition-timing-function：定义动画的速度曲线（这里是ease，慢快慢）
  • transition-delay：设置过渡开始前的延迟时间

在这个案例中，我们为卡片整体和作者名字分别设置了不同的过渡效果，当鼠标悬停在卡片上时，会平滑地改变背景色、阴影和文字颜色。

5. 注意事项

  • 不是所有的CSS属性都可以应用过渡效果，只有具有中间值的属性才可以（如颜色、尺寸、位置等）
  • display属性不能应用过渡效果，可以使用visibility或opacity代替
  • 过渡效果需要触发条件，如:hover、:active、:focus等伪类或JavaScript修改样式
  • 使用过渡时要注意性能，避免过多元素同时进行复杂过渡
  • 使用GPU加速的属性（如transform、opacity）可以获得更好的性能
